КАК ЗАПУСТИТЬ ИГРУ summer-game v2.0

1. Убедись, что у тебя установлен Python 3.
   Проверить можно командой в терминале:
   python --version

2. Скачай файл с игрой: v2.0 code.py

3. Открой терминал в папке, где лежит этот файл.

4. Запусти игру командой:
   python "v2.0 code.py"

5. Следуй инструкциям на экране.
   Вводи имя, возраст, деньги и делай выбор.

СОВЕТ
Если игра не запускается, попробуй заменить команду на:
   python3 "v2.0 code.py"

Приятной игры! ☀️
